# Multithreading Labs

Лабораторні роботи з багатопотоковості та паралельного програмування.

## Структура проекту

- `lab_1/` - Лабораторна робота №1

## Вимоги

- C++11 або вище
- CMake 3.0+
- Компілятор g++ (MinGW/MSYS2 для Windows)

## Збірка

### Використовуючи CMake

```bash
cd lab_1
mkdir build
cd build
cmake ..
cmake --build .
```

### Використовуючи g++ напряму

```bash
cd lab_1
g++ -o main main.cpp
```

## Запуск

```bash
./main
```
